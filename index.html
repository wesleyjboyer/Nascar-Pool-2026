NASCAR Pick'em Pool

This app syncs in real-time using Supabase — a free database so all your friends share the same data. Follow the 3 steps below, then you're done forever.

1Create a free Supabase project

Go to supabase.com → click Start your project → sign up and create a new project. Note your project's Region and choose any password. It takes about 30 seconds to provision.

2Run this SQL in your project

In your Supabase dashboard, go to SQL Editor (left sidebar) and paste & run the following:

create table participants (id text primary key, name text not null); create table races (id text primary key, name text not null, track text default '', race_date text default '', results jsonb default '[]'::jsonb); create table picks (id text primary key, pid text, rid text, drivers jsonb default '[]'::jsonb); create table custom_drivers (name text primary key, num text, bg text default '#333333', fg text default '#ffffff', removed boolean default false); alter table participants disable row level security; alter table races disable row level security; alter table picks disable row level security; alter table custom_drivers disable row level security;
3Paste your project credentials

In Supabase, go to Project Settings → API. Copy the Project URL and the anon public key.
